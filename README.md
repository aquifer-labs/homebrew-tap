# Aquifer Labs — Homebrew tap

```sh
brew install aquifer-labs/tap/artesian
```

Installs [**Artesian**](https://github.com/aquifer-labs/artesian) — the memory control plane for
agent loops — built from source with the `qdrant` and `llm` features. Provides the `artesian`,
`artesiand`, and `artesian-mcp` binaries. The Rust toolchain is pulled in automatically as a build
dependency.

To install the bleeding edge from `main`:

```sh
brew install --HEAD aquifer-labs/tap/artesian
```
